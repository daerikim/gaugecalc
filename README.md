게이지 계산만을 위한 게이지 계산기입니다. 나의 게이지, 도안 게이지, 도안 코수를 입력하고 계산하기를 누르면 자동으로 결과 값을 보여줍니다.

## 핵심 기능  Key Feature
- 저장된 공식에 따라 코수를 계산해줍니다.

## 사용 How To Use
- 박스에 써있는 대로 값을 적고 계산하기를 누릅니다.
